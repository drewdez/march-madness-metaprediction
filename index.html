2015 March Madness Metaprediction

Wherein Kagglers collectively predict the winners of the March Machine Learning Mania prediction contest (and, of course, the outcome of the NCAA Tournament itself).

The tables below were created by simulating the NCAA Tournament 10,000 times at each completed stage of the tournament, using the median Kaggle March Machine Learning Mania 2015 predictions as the true probabilities for all potential matchups.

Selecting a round below displays simulation results as of the start of the corresponding round. Select "Kaggle Results" to display the simulated results of the Kaggle competition, or "Tournament Results" to display the simulated results of the tournament.
